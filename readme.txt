== Installation ==
1. Upload the folder "hide-product-data-tabs" to "/wp-content/plugins/" path.
2. Activate the plugin through the "Plugins" menu in WordPress backend
3. After activating the plugin, product attribute will automatically remove Product Data Tabs.
4. Enjoy!
